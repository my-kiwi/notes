# Notes
Basic notes app PWA: https://my-kiwi.github.io/notes

## Install the PWA
### Apple iOS
1. Tap Share
2. Swipe up and tap "Add to Home Screen"
### Android
1. Tap the three-dots menu (top-right), 
2. Select "Add to Home screen"
3. Confirm by tapping "Add"

## Project setup
after cloning, run
- `nvm use`
- `npm i`
- `npm run dev`
